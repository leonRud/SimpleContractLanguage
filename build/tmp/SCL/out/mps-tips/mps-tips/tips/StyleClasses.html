Style classes provide an extension mechanism to describe that a particular style class inherits values of all style properties, which are not overridden explicitly.

  • A special style property apply can be used to copy values of all properties specified in the parent style class into our style class.
  • An apply-if variant is also available to apply a style property value conditionally.
  • The unapply property allows style classes to cease the effect of selected inherited properties.
  • A style class can be declared to dominate over some other style class or multiple classes and thus force its own properties to take precedence.
