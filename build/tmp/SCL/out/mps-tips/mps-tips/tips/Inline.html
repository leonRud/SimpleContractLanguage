The Inline Local Variable and Inline Method refactorings in BaseLanguage help you simplify complicated statements in your code by in-lining the referenced variables or the called methods into the place, where they are being used.

You can select (or position the cursor on) a local variable reference or a method call and press &shortcut:Inline; (Refactoring | Inline Local Variable or Refactoring | Inline Method in the pop-up menu).
